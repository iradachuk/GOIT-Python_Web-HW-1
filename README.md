# GOIT-Python_Web-HW-1

Репозиторій для домашнього завдання до модуля 1 (Прийоми об'єктно-орієнтованого програмування)

## **Завдання**

1. Намалюйте UML діаграму вашого випускного проекту "Персональний помічник" з минулого курсу. Для роботи можете скористатися безкоштовним [draw.io](https://app.diagrams.net/) або будь-яким іншим зручним для вас додатком.

2. Ваш попередній додаток зараз працює в консольному режимі та взаємодіє з користувачем у вигляді команд в консолі. Додаток треба розвивати і найчастіше змінюваною частиною додатку зазвичай є інтерфейс користувача (поки що це консоль). Модифікуйте код вашого додатку, щоб подання інформації користувачу (виведення карток з контактами користувача, нотатками, сторінка з інформацією про доступні команди) було легко змінити. Для цього треба описати абстрактний базовий клас для користувальницьких уявлень та конкретні реалізації, які успадковують базовий клас та реалізують консольний інтерфейс.
